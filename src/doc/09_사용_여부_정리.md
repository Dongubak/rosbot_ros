## 기능별 사용 여부

### ✅ 사용하는 것

#### 차동 구동 (Differential Drive)
- `diff_drive_controller` — 차동 구동 컨트롤러
- 설정: `rosbot_controller/config/rosbot/controllers.yaml`
- 수정 필요: 바퀴 치수 (wheel_separation, wheel_radius)

#### IMU 센서 통합
- `imu_sensor_broadcaster` — IMU 데이터 발행
- `RosbotImuSensor` 하드웨어 인터페이스 (또는 유사 구현)
- EKF에서 IMU 데이터 융합

#### EKF 로컬라이제이션
- `robot_localization` 패키지의 `ekf_node`
- 바퀴 오도메트리 + IMU 융합
- 설정: `rosbot_localization/config/ekf.yaml`

#### 레이저 필터
- `laser_filters` 패키지의 박스 필터
- 로봇 자신의 몸체 필터링
- 수정 필요: Cobra Flex 풋프린트에 맞게

#### 조이스틱 조종
- `joy_node` + `teleop_twist_joy`
- 설정: `rosbot_joy/config/joy.yaml`

#### 로봇 상태 발행
- `robot_state_publisher` — URDF + 관절 상태 → TF
- `joint_state_broadcaster` — 관절 상태 발행

### ❌ 사용하지 않는 것

#### 매니퓰레이터 관련

> [!WARNING] 이 코드들은 우리 프로젝트와 무관하다

| 파일/코드 | 이유 |
|-----------|------|
| `rosbot_controller/launch/manipulator.launch.py` | 매니퓰레이터 없음 |
| `rosbot_controller/config/rosbot_xl/controllers.yaml` (manipulator 섹션) | 매니퓰레이터 없음 |
| `rosbot_description/urdf/rosbot_xl/*.urdf.xacro` (manipulator 관련) | ROSbot XL 전용 |
| `open_manipulator_x_*` 외부 패키지 의존성 | 매니퓰레이터 없음 |
| `MANIPULATOR.md` 문서 | 매니퓰레이터 가이드 |
| Configuration: `manipulation`, `manipulation_pro` | 매니퓰레이터 구성 |

#### 옴니휠(메카넘 드라이브) 관련

> [!WARNING] 이 코드들은 우리 프로젝트와 무관하다

| 파일/코드 | 이유 |
|-----------|------|
| `mecanum_drive_controller` | 옴니휠 없음 |
| `rosbot_controller/config/rosbot_xl/controllers.yaml` (mecanum 섹션) | ROSbot XL 전용 |
| `rosbot_description/urdf/rosbot/wheel.urdf.xacro` (mecanum 분기) | 메카넘 URDF |
| Gazebo 메카넘 마찰 설정 (mu2, slip1, fdir1) | 메카넘 물리 시뮬레이션 |
| Launch 파라미터: `mecanum:=True` | 메카넘 활성화 옵션 |

#### ROSbot XL 전용 코드

| 파일/코드 | 이유 |
|-----------|------|

## 혼동하기 쉬운 부분

### 1. `mecanum:=False` vs 메카넘 미사용

launch 파일에 `mecanum` 파라미터가 있어도 걱정할 필요 없다.
`mecanum:=False` (기본값)로 실행하면 차동 구동이 사용된다.
**파라미터가 있다고 메카넘을 써야 하는 것이 아니다.**

### 2. ROSbot XL 설정 파일이 레포에 있는 이유

`rosbot_xl/` 폴더의 설정 파일들은 ROSbot XL용이다.
우리는 `rosbot/` (ROSbot 기본 모델) 기준의 설정만 참고하면 된다.

### 3. `arm_activate` 파라미터

```bash
bringup.launch.py arm_activate:=True  # 매니퓰레이터 활성화
```

이 파라미터는 우리와 무관하다. 기본값이 `False`이므로 자동으로 매니퓰레이터가 비활성화된다.

### 4. Configuration 옵션

우리는 `configuration:=basic`만 참고한다.
`manipulation`, `telepresence` 등은 ROSbot XL 특화 기능이다.
